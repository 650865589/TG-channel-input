# Telegram 本地自动化加群工具 (Telegram Auto Joiner)

这是一个基于 Python 和 `pyautogui` 的纯本地化 Telegram 自动加群/频道工具。它不依赖 Telegram API（因此不需要申请 API ID/HASH，也不会遇到网络受限的问题），而是直接利用你电脑上已登录的 **Telegram 官方桌面版客户端** 和 Windows 系统底层协议来进行自动化鼠标点击操作。

此外，本工具还会自动提取杂乱无章的文本中的有效链接，并在成功加入群组/频道后，**自动执行静音 (Mute)** 操作，避免被海量消息打扰。

## ✨ 核心特性

- **零 API 依赖**：直接驱动本地 Telegram 客户端，无需再去繁琐地申请和配置 API。
- **智能链接提取**：采用强大的正则表达式，无论你的 `links.txt` 中包含换行、逗号还是其他无关文本，都能精准提取出有效的加群链接（兼容标准链接、`joinchat`/`+` 私密链接、甚至纯文本 `@username`）。
- **全自动加群与静音**：能够区分公开频道和私密群组，并在成功加入后自动点击静音按钮。
- **安全防误触**：内置 `pyautogui` 安全机制，在程序运行过程中，如果需要紧急停止，**只需将鼠标快速滑动到屏幕的任意一个角落**，即可强行终止程序。

## 🛠️ 环境准备

1. **操作系统**：Windows (因为利用了 `tg://` 协议拉起应用)。
2. **Python 环境**：建议 Python 3.7 及以上。
3. **Telegram 客户端**：电脑上必须安装并登录了 [Telegram Desktop 官方桌面版](https://desktop.telegram.org/)。

## 📦 安装与配置

1. **安装依赖**
   在项目主目录下运行以下命令安装需要的库:
   ```bash
   pip install -r requirements.txt
   ```
   *(本质上只需要安装 `pyautogui` 即可)*

2. **准备链接数据**
   打开项目中的 `links.txt`，将你需要加入的链接全部粘贴进去。不需要特意整理格式，只要文本中包含诸如 `t.me/xxx`，`@username`，`https://t.me/joinchat/...` 等特征，脚本都会自动识别并去重。

3. **微调延迟 (可选)**
   打开 `main.py`，你可以在顶部看到配置区域：
   - `DELAY_UI_LOAD`: 唤起 Telegram 界面后等待其加载的秒数（默认 3 秒）。如果你的电脑或网速较慢，可以适当调大。
   - `DELAY_BETWEEN_JOINS`: 每次成功加入并静音后，处理下一个链接前的休息时间（默认 5 秒），防止请求过快。

## 🚀 如何运行

1. 在终端中运行脚本：
   ```bash
   python main.py
   ```

2. **互动引导录制坐标**：
   运行后，程序会输出中文提示，并给你 **5秒倒计时**，你需要根据提示把鼠标放在响应的按钮上：
   - **录制公开频道 加入按钮**：随便找一个未加入的公开频道，将鼠标悬停在底部的「Join / 加入」按钮上。
   - **录制私有群组 弹窗加入按钮**：点击任意私密链接调出弹窗，将鼠标悬停在弹窗的「Join Group / 加入群组」按钮上。（如果你文件中没有私密链接，这步会自动跳过）
   - **录制静音 (Mute) 按钮**：随便找一个已加入的群组/频道，将鼠标悬停在底部的「Mute / 静音」按钮上。

3. **开始自动化放手运行**：
   记录完坐标后，脚本会自动开始遍历所有的链接，自动拉起 Telegram 界面 -> 移动鼠标点击加入 -> 移动鼠标点击静音 -> 等待几秒 -> 处理下一个。

   > ⚠️ **强烈警告**：在开始全自动执行后，**请不要移动鼠标或操作键盘**！如果需要紧急叫停，请用力将鼠标甩到屏幕的左上、左下、右上或右下角触发安全机制！
